PHP version: 5.5(+)
Mysql/MariaDB Version: 4.0+
Mysql improved extension: Optional but highly recommended, can be changed in vibe_config.php
Apache with mod_rewrite or Nginx with access to configuration file, but you need to convert the .htaccess files online
cURL: For video details grabbing and importers

Ioncube loader: 4.4+
Register_globals OFF
Safe_mode OFF
mod_security OFF
exec() and shell_exec() Enabled For FFMPEG

B. File Structure - top

If you extract downloaded ZIP file you should have this or a similar structure in the CMS's folder for upload.

  • Cache Contains several cache storage folders.
  • Com contains the components of every page (pre-work before the template).
  • Lib folder contains functions and libraries required by the script and extras.
  • Tpl includes the folders for the themes/skins
  • Media folder will store the files after the ffmpeg conversion.
  • RawMedia folder will store the files before the ffmpeg conversion (directly from the upload).
  • Uploads folder will store avatars and misc.
  • Plugins Contains the folders for the plugins.

Installation - top

The installation method involves downloading our cms and uploading it to your webserver via your favorite FTP application.

  1. Download the cms archive to your computer and unzip it
  2. Using an FTP program, or your hosting control panel, upload the unzipped cms folder to your server's web folder.
  3. Edit vibe_config.php carefully, checking all options. Run /setup , make the required chmods, create an administrator (or more) .

The configuration file - top

Edit the file vibe_config.php in the product’s folder with your database details, site url (don’t forget the ending slash) and license key. We recommend Notepad++ setup to UTF-8 no-bom (under Encoding).

– Chmod 0555 cron.php (execute permissions).
– Add  cron.php to your system crons.
Documentation:

Step 5: FFMPEG configuration (optional)
– Chmod 0555 the file videocron.php

D. Troubleshooting - top

= I can't login or pages aren't working, what should I do now? =

Links/login/pages are not working?

– Make sure you upload and overwrite the .htaccess file in the PHPVibe root.
– Make sure you use Apache (else rewrite the rules for your server via a tool online) and that mod_rewrite works (AllowOverride check)
– Make sure if you upload it in a folder that you change the RewriteBase / to /your-folder-name in the main .htaccess file, and then under Administration -> Settings -> Permalinks you add your folder name as BasePath for the links.

= Can’t create accounts/administrators =

This is probably caused by the STRICT_TRANS_TABLES SQL mode defined MySql (Safe mode). Removing that setting and restarting MySQL should fix the problem.

= Dates are incorrect. =

Set the timezone to yours. 

/** Timezone (set your own) **/
date_default_timezone_set('Europe/Bucharest');

PHP : List of Supported Timezones

= Cannot upload large files =

Please see Server settings for large file uploads

= Uploaded video not working =

Uploaded video not working on PHPVibe / Missing Video/ FFMPEG debug

= Pages returning 404 from time to time, weird blockages on upload, sessions being killed =

Solution : Disable mod_security or other antivirus on the account.
